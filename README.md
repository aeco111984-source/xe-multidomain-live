# XE-Lite Multi-Domain FULL
One codebase → many domains. Includes full converter + optional tabs.

## Steps
1) Upload to GitHub
2) Vercel → Import Project → Deploy
3) Add your domains in Vercel
4) Adjust per-domain config in `config/site.config.json`

## Notes
- Converter uses free exchangerate.host baseline
- Charts/Live prices use TradingView widgets
- Optional tabs (calendar/infographics/widgets/news) on/off per domain
